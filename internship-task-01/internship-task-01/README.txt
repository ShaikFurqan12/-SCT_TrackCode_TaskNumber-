# Internship Task 01 - CSV Bar Chart

This project reads population-like data from a CSV file and generates a bar chart using matplotlib.

## Files
- `population_data.csv`: Mock CSV file used as dataset
- `chart_from_csv.py`: Python script to read the CSV and plot the bar chart
- `population_chart.png`: Output chart image (generated after running the script)

## How to Run
1. Install dependencies:
   pip install pandas matplotlib

2. Run the script in IDLE or any Python environment:
   python chart_from_csv.py
